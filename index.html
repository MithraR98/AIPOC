What is CLAUDE.md?
A Markdown fileA simple text file with formatting (like bold, headings) — no special software needed that lives in your project folder. Claude reads it automatically when you start working — like opening a project brief. It tells Claude everything about your project: what tools you're using, your design rules, and any important constraints.
